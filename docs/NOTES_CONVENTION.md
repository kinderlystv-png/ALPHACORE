# Конвенция записей: где твоё, где моё

## Визуальная дифференциация
Используем обязательные маркеры:

- `🟩 USER NOTES` — твои мысли, идеи, вводные, приоритеты.
- `🟦 ASSISTANT NOTES` — мои предложения, улучшения, структурирование, гипотезы.

## Правило применения
1. В каждом рабочем документе должны быть явно выделены оба блока.
2. Если данных от пользователя нет, блок `🟩 USER NOTES` остаётся с пометкой `TBD`.
3. Я не переписываю `🟩 USER NOTES`, а только структурирую рядом в `🟦 ASSISTANT NOTES`.
4. Любые спорные интерпретации фиксируются как гипотеза в `🟦 ASSISTANT NOTES`.

## Минимальный формат
```md
## 🟩 USER NOTES
- ...

## 🟦 ASSISTANT NOTES
- ...
```
