# Weekly Schedule (draft)

## 🟩 USER NOTES
- По средам обычно с **18:00 до 23:00** — репетиция на барабанах в гараже у друга, вчетвером, играете каверы.

## 🟦 ASSISTANT NOTES
- Это фиксированный личный блок, его не перегружаем рабочими задачами.

## Базовые правила недели
1. Среда 18:00–23:00 — **locked personal block** (барабаны).
2. Критичные рабочие задачи не ставим в этот интервал.
3. В день репетиции — только лёгкие операционные задачи до/после.

## Черновая недельная рамка
- **Пн:** стратегические и сложные задачи (Kinderly/HEYS)
- **Вт:** реализация + коммуникации
- **Ср:** до 18:00 рабочий блок, 18:00–23:00 барабаны
- **Чт:** follow-up и закрытие хвостов
- **Пт:** weekly review + план следующей недели
- **Сб/Вс:** по ситуации (семья/восстановление/лёгкие задачи)

## Категория приоритетов «Желание внедрить»
- **Сон:** отбой до 00:00, целевые 7 часов сна.
- **Бег:** через день по 60 минут, **кроме среды и четверга**.
- **Растяжка:** утром 10–20 минут, программу упражнений оформляем отдельно и можно донастраивать.

## Пространство для твоих идей
- Этот график — опорный, не жёсткий; твои предложения приоритетны и могут менять структуру недели.
- Изменения добавляем точечно (без бюрократии): что поменяли, почему, какой эффект ожидаем.

## Ограничения по бегу и восстановлению
- **Среда:** бег не ставим — вечером интенсивная репетиция на барабанах (высокая нагрузка).
- **Четверг:** после гаража бег не ставим; максимум — мягкая растяжка/восстановление для снижения крепатуры.
- Беговые слоты смещаем на **Пн / Вт / Сб** (или Пн / Пт / Вс по самочувствию).
