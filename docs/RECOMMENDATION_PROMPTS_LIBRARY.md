## 🟦 ASSISTANT NOTES
Ниже короткие промпты, уже пригодные для кнопки copy.

## 1) Kinderly — следующий шаг воронки
```text
Разверни следующий рычаг по Kinderly в 3 конкретные задачи:
- карта воронки,
- CTA и форма заявки,
- SLA ответа.

Обнови docs и предложи измеримые KPI на 7 дней.
```

## 2) HEYS — health floor на неделю
```text
Собери минимальный health floor на 7 дней:
- сон,
- бег (без среды/четверга),
- утренняя растяжка.

Сделай реалистичный план и добавь блок корректировок по самочувствию.
```

## 3) Agent review (короткий)
```text
Сделай краткий review:
1) что движется,
2) что буксует,
3) какой 1 следующий шаг с максимальным эффектом.

Формат: очень кратко, но с конкретикой.
```
