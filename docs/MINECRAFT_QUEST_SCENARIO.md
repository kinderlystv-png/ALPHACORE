# Сценарий квеста Minecraft (черновик v1)

## 🟩 USER NOTES
- Нужны квесты для детей в теме Minecraft с особым акцентом на Даню.

## 🟦 ASSISTANT NOTES
- Сценарий сделан модульным: можно сократить или расширить по числу детей и времени.

## Цель
- Создать динамичный, понятный и безопасный квест для группы детей.

## Этапы (4–6)
1. **Старт-миссия**: “Соберите ресурсы”
   - Дети получают карту/подсказки.
2. **Крафт-станция**
   - Мини-задание на сбор комбинаций.
3. **Защита базы**
   - Командное задание на скорость/внимание.
4. **Поиск “редкого блока”**
   - Поиск по подсказкам в фотозоне/пространстве.
5. **Босс-уровень (опционально)**
   - Финальное общее задание.
6. **Награждение**
   - Символические призы и фото.

## Акцент на Даню
- Отдельная “роль героя” в финале (ключ/артефакт/командир).
- 1 персональное задание по его интересу.

## Материалы
- Карточки заданий, маркеры, наклейки, жетоны
- Реквизит для зон/станций
- Призы участникам

## Роли
- Ведущий квеста
- Помощник по реквизиту
- Фото/видео координатор
- Ответственный за тайминг
