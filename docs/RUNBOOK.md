# ALPHACORE — Runbook (Template)

## Инциденты
1. Зафиксировать время начала и влияние
2. Проверить статус БД, кэша, очередей, внешних API
3. Откатить последний релиз при необходимости
4. Коммуникация в статус-канал каждые N минут
5. Postmortem в течение 24 часов

## Чеклист перед релизом
- [ ] Миграции проверены на staging
- [ ] Backward compatibility подтверждена
- [ ] Smoke tests пройдены
- [ ] Алерты и дашборды актуальны

## Чеклист после релиза
- [ ] 5xx/error rate в норме
- [ ] latency в пределах ожидаемого
- [ ] нет деградации бизнес-метрик
