2. **Окно встречи гостей**
   - Приветствие, ориентация, разогрев.
3. **Окно квеста**
   - Основные этапы + паузы.
4. **Окно торта и поздравления**
   - Вынос торта-крипера, фото, поздравления.
5. **Окно свободной активности**
   - Игры, фото, завершение.
6. **Окно закрытия**
   - Сбор площадки, финальный чеклист.
