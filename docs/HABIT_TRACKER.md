# Трекер привычек — ALPHACORE

## Концепция
Трекер привычек — центральный инструмент дашборда. Позволяет заводить ежедневные/еженедельные действия и отмечать их выполнение. Цель — видеть прогресс или регресс и поддерживать дисциплину.

## Текущие привычки (defaults)

| ID | Название | Частота | Активные дни | Категория |
|----|----------|---------|--------------|-----------|
| `sleep` | Сон до 00:00 | daily | все | health |
| `run` | Бег 60 мин | custom | Пн, Вт, Пт, Сб | health |
| `stretch` | Растяжка | daily | все | health |
| `projects_upd` | Апдейт проектов | custom | Пн, Чт | work |
| `review` | Weekly review | custom | Пт | work |
| `drums` | Барабаны | custom | Ср | personal |

## Как добавить/изменить привычку
Редактировать массив `DEFAULT_HABITS` в `src/lib/habits.ts`.

Каждая привычка:
```ts
{
  id: "unique_id",
  name: "Название",
  emoji: "🏃",
  frequency: "daily" | "custom",
  days?: [1, 2, 5],  // 0=Вс, 1=Пн, ..., 6=Сб
  category: "health" | "work" | "personal",
}
```

## Хранение данных
- **Ключ localStorage:** `alphacore_habits`
- **Формат записей:** JSON-объект, ключи `{habitId}:{YYYY-MM-DD}`, значения `true/false`
- При переходе на БД — заменить функции `load()`/`save()` в `src/lib/habits.ts`

## Визуальные элементы

### Progress Ring (SVG)
Круговой индикатор процента выполнения привычек за сегодня. Обновляется при каждом клике.

### Недельный Bar Chart
7 столбцов за последние 7 дней. Высота пропорциональна проценту выполнения:
- 100% — полный emerald
- 1–99% — полупрозрачный emerald
- 0% — zinc (тёмный)

Сегодняшний столбец выделен ring-подсветкой.

### Streak (серия)
Количество последовательных дней с полным выполнением всех активных привычек. Считается с вчерашнего дня назад, чтобы незавершённый сегодня не обнулял серию.
